# 🧠 Exercícios em C #

Coleção de **15 exercícios práticos em C# (Console Application)** desenvolvidos com o objetivo de fortalecer a lógica de programação e os fundamentos da linguagem.

---

## 🎯 Objetivo

Praticar conceitos essenciais de C#:

- Entrada e saída de dados  
- Conversão de tipos  
- Estruturas condicionais  
- Estruturas de repetição  
- Resolução de problemas  

---

## 🛠️ Tecnologias

- C#  
- .NET (Console Application)  
- .NET Fiddle  

---

## 📚 Exercícios

### 🔹 Nível 1 – Entrada e Saída
- Estoque médio  
- Conversor de moedas  
- Comissão de vendas  
- Cálculo de consumo  
- Conversor de temperatura  

---

### 🔹 Nível 2 – Condicionais
- Valor absoluto  
- Ordenação de números  
- Validação de faixa  
- Menu com switch  
- Classificação de triângulos  

---

### 🔹 Nível 3 – Repetição
- Contagem regressiva  
- Tabuada automática  
- Soma de pares e ímpares  
- Monitor de poluição  
- Jogo de adivinhação  

---

## 🚀 Execução

1. Abrir o projeto como **Console Application**  
2. Executar no Visual Studio ou .NET Fiddle  
3. Interagir via terminal  

---

## 📌 Observações

- Foco em lógica, não em interface  
- Exercícios progressivos (básico → intermediário)  
- Base para projetos maiores
